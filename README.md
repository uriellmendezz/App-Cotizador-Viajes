# Cotizador Viajes One Trip

Aplicación web interactiva para la creación, optimización y exportación de cotizaciones turísticas del equipo de One Trip.

## Tecnologías
* **Backend:** FastAPI (Python)
* **Frontend:** Vanilla HTML, CSS (Tailwind) y Javascript
* **Base de Datos:** Supabase (PostgreSQL)
* **Generación de PDF:** WeasyPrint
* **Motor de IA:** Groq API (Llama 3)
* **Contenedor:** Docker
